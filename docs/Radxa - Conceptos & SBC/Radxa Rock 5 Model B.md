## <font color="#de7802">2. Especificaciones Principales</font>

Es una computadora de placa unica (*board computer*) basada en el RK3588 chipset (para acceder a la documentación extensiva sobre este chipset cualquier cosa encontré este [link](https://github.com/axlrose/rkdocs) muy bueno). Esta SBC (*single board computer*) nos permite trabajar sobre varias cosas como:
- Controlador Robot
- Agente local de IA
- Nodo Blockchain
- Servidor Privado
- Entre otros...

Contiene varios puertos USB y HDMI para conectarlos a dispositivos externos y potenciar su uso, como con camaras. Además, tiene una *AI accelerator unit* que permite procesar gráficos más rapidamente. Además, puede correr varios OS desde Android hasta Debian/Ubuntu.
